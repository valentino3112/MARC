# CDataframe-CAO-GONCALVES

## Introduction
Ce projet a pour objectif d'améliorer nos connaisances sur de nombreuses structures de données en C (les arbres, les piles et les files). Pour cela on imagine un rover Marsien qui doit se déplacer pour rejoindre une base. On doit donc crée un programme qui permet de déterminer le chemin le plus court pour le rover.
## Structure
Il est composé de six programmes principaux :

- Tout d'abord : 'main.c' qui a pour objet de gérer les échanges entre l'utilisateur et la machine. Il est relié à tous les autres programmes.
- Puis : 'arbre.c' qui contient toutes les fonctions permettant de modifier les arbres et les valeurs qu'ils contiennent.
- 'loc.c' qui contient les fonctions relatives à la localisation des valeurs dans les arbres.
- 'moves.c' qui contient les fonctions relatives aux déplacements du rover.
- 'queue.c' qui contient les fonctions relatives aux piles.
- Enfin, 'time_calcul.c' qui contient les fonctions relatives au calcul du temps d'exécution des fonctions.

Enfin, il faut noter que tous ces fichiers, à l'exception de 'main.c', ont leurs parallèles en .h qui contiennent les prototypes de leurs fonctions.

## Principales fonctionnalités de votre application

- Remplissage de l'arbres avec les différents mouvements possibles du rover.
- Affichage de l'entièreté de l'arbre.
- Recherche de la valeur la plus proche de la base.
- Analyse de la complexité des fonctions
